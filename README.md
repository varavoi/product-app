## О проекте

SPA приложение для управления карточками товаров с возможностью просмотра, поиска, фильтрации и управления избранными товарами.

## Основной функционал

- **Список товаров** - просмотр всех товаров с пагинацией
- **Избранное** - добавление/удаление товаров в избранное
- **Поиск** - поиск товаров по названию и описанию
- **Удаление** - возможность удалять товары
- **Создание** - форма добавления новых товаров
- **Адаптивность** - корректное отображение на всех устройствах
- **Сохранение** - все данные сохраняются в localStorage
